# H-UI
 快应用H-UI组件库开源
## 简介
 h-ui是一套为华为快应用开发者量身打制的UI组件库，以传统html5标签为基础，遵循主流前端框架样式命名习惯，对华为快应用原生组件二次封装而成，以实现快应用“全组件式开发”为目标，让快应用开发就像搭积木！
 
**快应用H-UI组件库承诺**
将持续推出更多精彩、精良的快应用组件。对原有组件会持续维护、优化升级，并且优化后的组件尽可能向前兼容，不影响已经使用了该组件的快应用。界面精美、功能强大、使用方便灵活以及运行时的高性能将是我们坚持的开发原则。H-UI出品，必属精品！
